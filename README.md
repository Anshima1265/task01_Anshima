# 🤖 Smart AI Chatbot

## 📌 Project Overview

Smart AI Chatbot is a rule-based chatbot developed using **Python** and **Tkinter**. The chatbot interacts with users through a graphical user interface (GUI) and responds to predefined commands using conditional logic (`if-elif-else` statements).

This project was developed as part of the **DecodeLabs Artificial Intelligence Internship - Project 1 (Rule-Based AI Chatbot)**.

---

* Understand the fundamentals of Artificial Intelligence.
* Implement rule-based decision-making.
* Develop a user-friendly chatbot interface.
* Learn Python GUI development using Tkinter.

## ▶️ How to Run

### Step 1: Install Python

Download and install Python from:
https://www.python.org/downloads/

Verify installation:

```bash
python --version
```

### Step 2: Download the Project

Save the file:

```text
smart_ai_chatbot.py
```

on your computer.

### Step 3: Run the Application

Open Command Prompt or VS Code Terminal and execute:

```bash
python smart_ai_chatbot.py
```
